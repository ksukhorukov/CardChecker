# Card Checker
## Intro

Validates credit cards according to Luhn algorithm and determines it's type

## Supported Types

* Visa
* Masteracrd
* AMEX
* Discover
* SberBank
* AlfaBank
* MIR

### Usage

```
gem install bundler
bundle
chmod +x app.rb
rspec
./app.rb
```

Enjoy!

### Ruby Version

```3.0.0```
